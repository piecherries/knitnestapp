# 📖 About Knit Nest

Knit Nest is an educational mobile application developed using **Flutter** and **Firebase**. The application aims to provide an interactive platform for beginners who want to learn crochet (knitting) in a structured and enjoyable way.

Users can watch tutorial videos, unlock learning modules, earn rewards, save crochet project notes, and track their learning progress.

This application was developed as the final project for the **Design, Develop, and Deploy Mobile Application** course.

---

# ✨ Main Features

- 🔐 User Authentication (Login, Register, Reset Password)
- 📚 Crochet Learning Modules
- 🎥 YouTube Video Tutorials
- 💰 Coin & Reward System
- 🔓 Unlock Premium Modules
- ⭐ Favorite Modules
- 📈 Learning Progress Tracking
- 📝 Crochet Notes
- 🎁 Daily Rewards
- 👤 User Profile
- 📖 Learning History

# 📖 How to Use

1. Register a new account or log in.
2. Explore the available crochet learning modules.
3. Unlock premium modules using coins.
4. Watch tutorial videos.
5. Complete modules to earn rewards.
6. Save crochet project notes.
7. Track your learning progress from your profile.

---

# 🎯 Application Purpose

Knit Nest was developed to make crochet learning easier and more engaging by providing structured learning materials, video tutorials, progress tracking, and a reward system within a single mobile application.
